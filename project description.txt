## ğŸš€ Quick Start

### 1. Environment Setup

```bash

# Activate the environment

```

### 2. Configuration

Copy the environment template and add your API keys:

```bash
cp .env.example .env
```

Edit `.env` and add your OpenAI API key:

- NEWSAPI_KEY  
- CRYPTOPANIC_API_KEY

**Web Dashboard (Recommended):**
```bash

```

**CLI Mode:**
```bash
python app.py --cli
```

**Sample Mission:**
```bash

```
